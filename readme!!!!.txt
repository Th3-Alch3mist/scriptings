Make a database and make a table named "paste" (without quotes).
Create 6 columns namely(in order) :
id (type : INT, primary key, auto increment)
uid (type : varchar, size : 13)
title (type : varchar, size : 100)
language (type : varchar, size : 20)
pastecode (type : text)

Edit the file pdoconnect.php and insert your database name, username and password.

Edit index.php, viewcode.php and search.php, replace yourwebsite.com with your link.

Upload and use it.
